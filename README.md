# 部署memos.top

>[!tip] 
 提示
 适配v0.22.0 以及以上版本

环境变量:

```
MEMOS_HOST=https://memos.ee   # memos地址
MEMOS_USER_ID=1       # 用户ID
MEMOS_LIMIT=20       # 每页显示数量
MEMOS_TITTLE=提剑追梦 # 标题
MEMOS_HEAD=          # 头部支持HTML代码
MEMOS_MENU=          # 菜单支持HTML代码
MEMOS_ICON=          # 图标
MEMOS_FOOTER=        #底部支持HTML代码
```
